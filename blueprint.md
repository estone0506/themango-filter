# The Mango Filter - Browser Extension

## 개요
브라우저에서 사용자가 지정한 키워드를 관리하고 필터링할 수 있도록 돕는 확장 프로그램입니다.

## 주요 기능
- **필터 키워드 관리:** 사용자가 차단하거나 강조하고 싶은 키워드를 추가/삭제할 수 있습니다.
- **데이터 보관:** `chrome.storage.sync`를 사용하여 브라우저 로그인을 통해 기기간 필터 목록이 동기화됩니다.
- **현대적인 UI:** 망고 컬러 테마의 깔끔하고 직관적인 디자인을 제공합니다.

## 프로젝트 구조
- `manifest.json`: 확장 프로그램 메타데이터 및 권한 설정.
- `index.html`: 팝업 창 UI 구조.
- `style.css`: 현대적인 디자인 및 반응형 스타일.
- `main.js`: 필터 추가/삭제/저장 로직.
- `icons/`: 확장 프로그램 아이콘 폴더.

## 향후 계획
- **Content Script 추가:** 실제 웹페이지(예: 네이버, 구글 등)에서 해당 키워드가 포함된 요소를 숨기는 기능 추가.
- **정규표현식 지원:** 더 강력한 필터링을 위한 Regex 지원.
- **다크 모드 지원:** 사용자 편의를 위한 테마 선택 기능.
