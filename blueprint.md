# The Mango Filter - 더망고 전용 확장 프로그램

## 개요
더망고 대량등록 프로그램(`tmg4084.mycafe24.com`) 관리자 페이지에서 필터 데이터를 수집하고 리스트를 관리하는 기능을 제공합니다.

## 구현된 기능
- **Content Script 적용:** 더망고 관리자 페이지 내의 HTML 요소에 접근 가능.
- **데이터 수집 인터페이스:** 팝업창에서 '수집하기' 버튼을 누르면 페이지의 테이블 데이터를 10개 가져옴.
- **리스트 필터링 (예시):** 페이지의 항목 중 상위 10개만 노출하도록 제어하는 로직 포함.

## 프로젝트 구조 업데이트
- `manifest.json`: `host_permissions` 및 `content_scripts` 설정 추가.
- `content.js`: (New) 실제 웹페이지 내에서 동작하며 데이터를 추출하고 DOM을 조작함.
- `main.js`: 팝업 UI와 `content.js` 사이의 통신 로직 추가.

## 향후 과제 (정확한 구현을 위해 필요한 정보)
- **정확한 Selector 분석:** 현재 `table tr`로 되어 있는 선택자를 실제 더망고 페이지의 상품 리스트 ID나 Class명으로 변경해야 함.
- **로그인 처리:** 관리자 페이지이므로 세션 유지 및 로그인 상태 확인 로직 필요.
