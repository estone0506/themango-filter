# The Mango Filter - 더망고 전용 확장 프로그램

## 개요
더망고 대량등록 프로그램(`tmg4084.mycafe24.com`) 관리자 페이지에서 필터 데이터를 수집하고, 수집된 데이터를 바탕으로 마켓별 상품 삭제를 자동화하는 기능을 제공합니다.

## 구현된 기능
- **Content Script 적용:** 더망고 관리자 페이지 내의 HTML 요소에 직접 접근 및 조작 가능.
- **필터 데이터 수집:** 필터 삭제 페이지(`pmode=filter_delete`)에서 필터 ID와 이름을 수집하여 팝업 리스트에 표시.
- **마켓 실시간 동기화:** 팝업에서 선택한 마켓(쿠팡, G마켓, 11번가 등) 상태를 페이지 내의 체크박스와 실시간으로 연동.
- **자동 삭제 프로세스:**
    - **마켓 삭제 시작:** 페이지에서 현재 선택된 상품들에 대해 마켓 삭제를 실행.
    - **마켓삭제시작(검색결과모든상품):** 검색 결과에 해당하는 모든 상품에 대해 마켓 삭제를 실행 (일괄 삭제).
- **편의 기능:** 전체 선택/해제, 필터 목록 저장 및 삭제 기능 제공.

## 프로젝트 구조
- `manifest.json`: `host_permissions` 및 `content_scripts` 설정.
- `content.js`: 페이지 내 데이터 추출, 체크박스 동기화 및 삭제 스크립트 주입/실행.
- `popup.js`: 팝업 UI 제어, 상태 저장, `content.js`와의 메시지 통신 관리.
- `styles.css`: 모던하고 직관적인 팝업 UI 디자인.
